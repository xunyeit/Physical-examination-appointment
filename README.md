### 一、产品概述
**“****体检预约****”是一款专注于****健康体检服务在线预订与管理****的数字化医疗服务平台，致力于为个人及企业提供便捷、高效、透明的一站式体检解决方案。平台整合多家正规医疗机构的体检资源，提供标准化体检套餐、个性化自选项目、智能预约排期、电子报告查询等全流程服务，实现“****线上选检、线下体检、数据可查****”的智慧健康管理闭环。**

**该应用面向关注健康的普通人群、企业员工、中老年家庭用户以及有特定健康需求的个体（如入职体检、婚前检查、慢性病筛查），支持按需选择体检项目、灵活安排时间、在线支付并生成电子订单，大幅提升了传统体检流程的效率与体验。**

**“体检预约”不仅是一个订体检的工具，更是一个推动主动健康管理、预防医学普及和医疗资源优化配置的重要载体，助力实现“从治病到防病”的健康转型。**

### 二、核心功能
1. **套餐升级****  
****用户可在基础体检套餐基础上，根据自身需求添加专项检查项目（如肿瘤标志物、心血管评估、骨密度检测等），实现个性化定制。**
2. **自选检查****  
****支持用户自由组合体检项目，自主选择科室与检查内容，满足不同年龄、性别、职业或健康风险人群的多样化需求。**
3. **体检预约****  
****提供医院/机构列表及可用时间段选择，支持在线预约具体日期与时段，避免现场排队等候，提升就诊效率。**
4. **我的订单****  
****查看所有已创建的体检订单，包括待支付、已预约、已完成等状态，支持订单详情查询、取消操作与售后服务申请。**
5. **未支付****  
****展示尚未完成付款的订单，提醒用户及时支付，防止预约失效。**
6. **已预约****  
****列出已成功预约但尚未体检的订单，支持查看预约信息、修改时间、打印电子凭证等操作。**
7. **已完成****  
****记录已完成的体检记录，支持查看体检结果、报告下载、健康建议等后续服务内容。**
8. **关于我们****  
****展示平台运营主体、合作医院、服务承诺、隐私政策、联系方式等信息，增强用户信任感与品牌认知。**
9. **推荐套餐****  
****（主界面功能）根据用户画像（如年龄、性别、职业）智能推荐适合的体检套餐，如“关爱父母”“女性专区”“入职体检”“高端定制款”等。**
10. **DIY自选检查****  
****提供可视化选项卡，用户可逐项勾选所需检查项目，系统自动计算总价并生成专属方案。**
11. **专业售后保障****  
****（隐含功能）提供体检后报告解读、异常指标提醒、医生咨询、复诊建议等增值服务，确保健康管理持续落地。**
12. **电子报告查询****  
****体检完成后，用户可通过平台查看高清电子版体检报告，支持导出PDF或分享给家人医生。**
13. **多账号管理****  
****支持为家庭成员（如父母、子女）建立独立档案，统一管理全家人的体检计划与历史记录。**
14. **企业团检支持****  
****（可扩展功能）为企业客户提供批量预约、员工名单导入、集中体检安排、数据统计分析等团体服务。**
15. **地图导航****  
****（隐含功能）点击预约医院后，自动跳转至地图应用，提供路线导航与交通提示，方便用户准时到达。**

### 三、产品价值
**用户价值：让体检更简单、更科学、更有温度**

+ **省时省力****：无需线下排队挂号，一键预约、在线支付，节省大量时间和精力。**
+ **透明可控****：所有项目明码标价，无隐形消费，用户可自主选择，避免“被推销”。**
+ **个性化服务****：根据不同人群设计专属套餐，满足差异化健康需求，提升体检精准度。**
+ **全程追踪****：从预约到报告出具，全过程可视化管理，便于用户掌握进度与结果。**
+ **家庭健康管理：支持多人账户绑定，实现全家健康档案统一管理，特别适合中老年群体照护。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

## 运维环境

+ **2 核 2 G 100MB 3M带宽 为最低配置，建议 2 核 4G 100Mb 5M带宽。**
+ **Node.js 14x**
+ **Java1.8**
+ **Mysql 5.7**
+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
